## Available Guides

\*   **Getting Started & App Management Guide:** This guide helps set up a developer account, create and manage applications, and understand client credentials.

\*   **OAuth 2.0 Authentication Guide:** This guide details the process of authenticating users and obtaining access tokens using the OAuth 2.0 protocol.

\*   **Refreshing Access Tokens Guide:** This guide explains how to refresh expired access tokens to maintain continuous access to user data.

\*   **Revoking User Access Guide:** This guide provides instructions on how to revoke a user's access token to disable an integration.

\*   **Retrieving User Cycle Data Guide:** This guide focuses on fetching a user's physiological cycle data from the WHOOP API.

\*   **Retrieving User Recovery Data Guide:** This guide explains how to retrieve a user's daily recovery scores and related metrics.

\*   **Retrieving User Sleep Data Guide:** This guide details how to fetch a user's sleep performance, stages, and sleep needs.

\*   **Retrieving User Workout Data Guide:** This guide covers accessing a user's workout activities, including strain, heart rate, and sport-specific details.

\*   **Retrieving User Profile & Body Measurements Guide:** This guide explains how to fetch basic user profile information and body measurements.

\*   **API Pagination Guide:** This guide provides instructions on how to paginate through large collections of data from the WHOOP API.

\*   **Webhooks Integration Guide:** This guide details how to set up, receive, validate, and process real-time event notifications via webhooks.

\*   **API Rate Limiting Guide:** This guide explains the API rate limits and how to interpret rate limit headers in responses.

\*   **App Approval Process Guide:** This guide outlines the steps required to submit an application for approval to launch to all WHOOP members.

\*   **Design & Brand Guidelines Guide:** This guide provides resources and instructions for adhering to WHOOP's design and brand standards.

\*   **WHOOP Core Concepts Guide:** This guide introduces the fundamental concepts of Strain, Recovery, and Sleep as measured by WHOOP.

\*   **Developer Support & FAQ Guide:** This guide provides information on how to get support and answers to frequently asked questions.
